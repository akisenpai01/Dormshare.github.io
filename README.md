# DormShare — P2P Campus Network

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/akisenpai01/Dormshare.github.io)

This repository contains the full "Digital Flagship" web interface and distributed Spring Boot backend for the DormShare platform.

## 🚀 One-Click Cloud Hosting
To fix all "Failed to Fetch" errors, host the backend in the cloud:
1. **Click the Button above.**
2. When prompted for `MONGODB_URI`, paste your [MongoDB Atlas](https://www.mongodb.com/cloud/atlas/register) connection string.
3. Once the deployment finishes, the backend is live globally! Client and infrastructure.
